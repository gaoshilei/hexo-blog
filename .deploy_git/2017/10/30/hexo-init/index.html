  1. 给博客目录赋权，让 nginx 用户拥有读写权限
  2. 让 nginx 以 root 用户运行

我采用第二种方式，修改 nginx 的配置文件

webhooks 在 github 对应仓库直接设置就行，重点是服务器的接收和相应的操作。
